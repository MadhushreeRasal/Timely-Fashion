# Timely - Fashion

A fashion and accessories themed product webpage built using HTML and CSS.  
This project was created to practice CSS Grid layouts, Flexbox positioning, hover effects, and multi-section webpage design while building a visually rich fashion product interface.

## Live Demo

https://madhushreerasal.github.io/Timely-Fashion/

## Features

- Fixed top navigation bar
- Sticky category menu
- CSS Grid based webpage layout
- Flexbox image gallery
- Fashion product showcase
- Interactive hover effects
- Customer review section
- User profile/account panel
- Product recommendation cards
- Custom typography using Google Fonts
- Remix Icon integration

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Google Fonts
- Remix Icons

## What I Practiced

- Creating complex layouts using CSS Grid
- Combining Grid and Flexbox together
- Sticky and fixed positioning
- Designing product galleries
- Building responsive-style content structures
- Styling cards, buttons, and reviews
- Working with imported fonts and icons
- Organizing large CSS sections

## Project Overview

The webpage represents a fictional fashion and accessories brand called **Timely - Fashion**.  
It includes:

- A scrolling promotional banner
- Navigation categories
- Featured luxury watch section
- Product specifications
- Customer reviews and feedback form
- User account/profile area
- Popular product recommendations
- Styled footer with social icons

The main goal of this project was to strengthen layout and styling skills through a large multi-section webpage.

## Notes

This is a static frontend practice project.

The project currently does not include:
- JavaScript functionality
- Backend/database integration
- Real cart system
- Authentication system
- Functional forms
